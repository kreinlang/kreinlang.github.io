Socket

This module uses the Berkeley sockets interface (BSD) to create socket connections, they are created by C wrapped from D and called from krein, so they are compatible to any language's socket connection.

Notice from this module.

Created sockets are not powerful enough to connect to javascript's webSocket but this will be made possible in the coming updates so watch out for the coming updates and mostly upgrades.
