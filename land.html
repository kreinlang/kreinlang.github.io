Expectation

  • Krein tries to keep its size as small as possible expecting you to get a copy of it incase you are trying to package your appliactions.
  • For now, the krein binaries are available for linux and windows only, but expect a Mac os, BSD and other Operating systems in less time to come also count mobile OS-es.
  • Built with D, memory management is managed full by Dlang's garbage collectors to allocate memory and part of the task is custom to have the best results.
  • Object Orientation is the order of the day here, everything is an object much like Python and Javascript. Less Datatypes are provided but your can create more customly using classes.
  • Support for large code bases is enabled using modules whereyou can write code in different files or folders but still access it.

Reviews

  • We have removed weak typing in the language, now you are required to enter the right params in a function otherwise it will error.
  • import keyword is much more self descriptive than use, so we have changed to import for better syntax and readability.
  • Random module is now atleast stable, many functions are added to make work easier so now it can even go in production.

Take note

  • Krein is in early access, many bugs are expected, less support from team and platforms. But the team is working on it so stability is soon comming.
  • Support for some platforms like Mac Os is not available yet. Probably due to lack of gadgets working with those platforms, if you can help Contact me.
