Console

This module is very essential because it deals with the interpreter core functions that are used to check datatypes states print there info to the terminal. It is even imported directly to the runtime for you.

List of functions in the console module.

Since console is imported for you, you are not expected to re-import it unless otherwise but even you will still be accessing the same pointer int memory.
